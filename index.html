Политика конфиденциальности

Настоящая Политика конфиденциальности (далее — «Политика») мобильного приложения «FinMind» (далее — «Приложение») действует в отношении той информации, которую ООО Специализированный «FinMind» (далее — «Общество») может получить с устройства Пользователя во время использования им Приложения.

1. Термины и определения

Приложение — приложение «FinMind» для мобильных операционных систем Apple iOS, Android.
Пользователь — физическое лицо, надлежащим образом присоединившееся к настоящему Соглашению для использования Приложения.

2. Общие положения

Использование Приложения означает безоговорочное согласие Пользователя с настоящей Политикой. В случае несогласия с Политикой пользователь должен воздержаться от использования Приложения.

Настоящая Политика применима только к Приложению. Общество не несет ответственности за информацию, переданную третьим сторонам по внешним ссылкам.

Общество может обновлять Политику, размещая новую редакцию в Приложении или на сайте.

3. Информация, которую мы собираем

  • Номер телефона — для регистрации и уведомлений.
  • Геолокация — для отображения ближайших сервисов.
  • Фото с камеры — для загрузки документов, профиля и оплаты.
  • Модель устройства и версия ОС — для улучшения стабильности.
  • IP-адрес и точка подключения — для безопасности.
  • SMS от Общества — для автоматической обработки в приложении.
  • Микрофон — для звонков в контактный центр.
  • Антивирус — защита от угроз, в том числе:
    • проверка файлов, сертификатов, сайтов;
    • обнаружение атак и уязвимостей;
    • улучшение качества работы и обновлений;
    • защита от фишинга и подмены сетевых ресурсов.

4. Условия обработки информации

  • Обрабатывается только информация, описанная выше, и только для указанных целей.
  • Данные антивируса не идентифицируют конкретное лицо.
  • Общество принимает меры для защиты информации от неправомерного доступа.
  • Могут привлекаться партнеры по соглашению о конфиденциальности.
  • Информация хранится в течение срока договора и до 5 лет после.
  • Информация может быть передана государственным органам по закону.

5. Дополнительная аналитика

Общество вправе использовать обезличенную статистику и cookie для анализа и улучшения работы Приложения.
