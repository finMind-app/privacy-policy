Политика конфиденциальности

Это тестовая политика конфиденциальности для демонстрации размещения через GitHub Pages. Она не является юридически значимым документом.

1. Сбор данных

Приложение не собирает и не передаёт никакие персональные данные пользователя на сторонние серверы.

Вся информация (доходы, расходы, заметки и другие данные) вводится пользователем вручную и сохраняется локально на устройстве.

2. Хранение данных

Все данные хранятся исключительно на устройстве пользователя и не покидают его пределы. При удалении приложения данные также удаляются.

3. Использование аналитики

Приложение не использует сторонние аналитические сервисы (например, Google Analytics, Firebase и т.д.).

4. Искусственный интеллект

Если в приложении используется искусственный интеллект, он работает исключительно на устройстве (on-device) и не передаёт данные на сервер.

5. Контакты

По вопросам, связанным с политикой конфиденциальности, вы можете связаться с разработчиком по электронной почте: emilsvetlichnyy@internet.ru
